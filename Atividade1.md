#  Atividade: Deploy de Aplicação Web com Múltiplas Telas no GitHub Pages

## Objetivo
Desenvolver uma aplicação web simples utilizando **HTML, CSS e JavaScript**, com navegação entre telas e publicação online.

---

##  Descrição da Atividade

Você deverá criar uma aplicação contendo **três telas principais**:

### 1. Tela Inicial (Landing Page)
- Deve conter:
  - Uma imagem de destaque
  - Um botão (ex: “Entrar” ou “Começar”)
- Ao clicar no botão, o usuário deve ser direcionado para a tela de login.

---

### 2. Tela de Login
- Deve conter:
  - Campo de usuário (ou e-mail)
  - Campo de senha
  - Botão de login
- Implementar validação simples em JavaScript:
  - Os campos não podem estar vazios
- Após login válido, o usuário deve ser redirecionado para a tela Home.

---

### 3. Tela Home
- Deve conter:
  - Um título de boas-vindas
  - Uma galeria de imagens (**mínimo de 3 imagens**)
- A galeria pode ser feita utilizando:
  - Grid
  - Flexbox
  - Ou um carrossel simples

---

##  Requisitos Técnicos
- Utilizar apenas:
  - HTML
  - CSS
  - JavaScript
- A navegação entre telas pode ser feita com:
  - Múltiplos arquivos HTML **ou**
  - Manipulação de DOM (SPA simples)
- O layout deve ser **responsivo** (adaptável a diferentes dispositivos)

---

##  Publicação (Deploy)
O projeto deve ser publicado utilizando **GitHub Pages**.

### Passos sugeridos:
1. Criar um repositório no GitHub
2. Enviar (push) dos arquivos do projeto
3. Acessar **Settings > Pages**
4. Selecionar a branch principal (main)
5. Obter o link gerado da aplicação
