#  Atividade Deploy com React — Interface com Componentes



Desenvolva uma aplicação utilizando **React** que contenha os seguintes elementos:

- Um **Header**
- **3 Cards com imagens**
- Um **Botão** (criado como componente reutilizável)

A aplicação deve apresentar uma interface organizada, com os componentes bem estruturados e reutilizáveis.

---

##  Requisitos

- Utilizar **componentização** (cada elemento deve ser um componente separado)
- Organizar o projeto em pastas (ex: `components`, `assets`)
- Utilizar imagens nos cards
- Criar um botão como componente independente
- Os cards devem estar alinhados na tela (layout organizado)
- Aplicar estilização (CSS)

---

##  Deploy

Após finalizar o projeto:

1. Realizar o deploy da aplicação utilizando o **Vercel**
2. Copiar o link gerado
